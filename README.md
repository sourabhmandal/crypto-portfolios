# crypto-portfolios

## Running backend
1. Run `npm install`
2. Add following env variables in `.env` file
```
MONGO_URI="mongodb://127.0.0.1:27017/local?directConnection=true&serverSelectionTimeoutMS=2000"
MONGO_DB_NAME="local"
```
3. Run `npm run dev`

## Running frontend
1. Run `npm install`
3. Run `npm run dev`
